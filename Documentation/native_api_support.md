# Native Render API support



## Native Render API

+ OpenGL(core 3.3)
+ OpenGLES(3)
+ DX11


## Platform

 Platform   | Native API                      | Achieve
------------|---------------------------------|---------------
 Windows    | DX11, OpenGL                    | DX11 origin, glfw
 Linux      | OpenGL                          | glfw
 Android    | OpenGLES                        | glfw
